HƯỚNG DẪN DỊCH MANHUNT 2


 

 

A. GIỚI THIỆU

 

Manhunt 2, một cái tên gây tranh cãi khá nhiều ở thời điểm mà nó ra mắt. Bạo lực, dung tục, tra tấn, máu me, tình dục… và còn nhiều thứ nữa được Rockstar gói gọn trong một game với tràn đầy những cảnh tượng kinh hãi khiến người chơi phải kinh hoàng khi trải nghiệm nó.

 

Là một game thủ lâu năm, không lạ gì mình cũng đã trải nghiệm và chơi thử con hàng này vài lần. Từ những bản PS2, PSP, cho đến PC. Và nhận thấy sao không thử nghiên cứu cách dịch nó xem sao.

 

Nói là làm, tôi bắt tay vào làm ngay và nhận ra rằng. Nó không dễ thực hiện chút nào. Từ khâu font đến text con hàng này thuộc loại khó nhằn ghê ghớm. Cũng may, trước tôi đã từng có cao thủ Matran999 nghiên cứu con hàng này trước rồi. Và đã tạo ra bộ tool font chia sẽ công khai.

 

Trong bộ tool này có cách nghiên cứu font map của nó, nhờ những thông tin này tôi đã tạo ra một bộ font In Hoa tiếng việt đầy đủ. Tại sao không dùng chữ thường ư? Có nhiều lý do lắm, chủ yếu là game không hỗ trợ quá nhiều ký tự để giúp thể hiện tiếng việt in thường, nên tôi phải chuyển hết sang in hoa để dễ nhìn.

 

Đồng thời, encode text của game này là UTF-16, thêm một đống pointer rối nhằn ở các file GXT nữa. Tuy rằng trên mạng có share những tool xuất text của game này, nhưng không may nó không phù hợp. 

  • Những tool này chỉ cho nhập và xuất mỗi từng file (Mất thời gian ghê gớm)
  • Tool này không hỗ trợ để nhập text thể hiện tiếng việt trong game.
  • Tool của người khác khó làm thay đổi hay chỉnh sửa bất cứ thứ gì.

Vì thế mìnhh đã tự nghiên cứu và viết tool riêng cho dòng game này. Cho phép nhập và xuất text hàng loạt các file GXT của game.

 

Dài dòng đủ rồi, bắt đầu vào việc chính thôi.

C. HƯỚNG DẪN DỊCH

 

Giờ bắt tay vào làm thôi. Dễ thôi, vì những thứ khó nhất mình đã làm hết cả rồi. Các bạn chỉ cần làm theo các bước là xong.

 

Bước 1 - Tải Dữ Liệu Việt Hóa

 

Các bạn tải file nén dữ liệu này về máy:

LinkDown: Link Google Drive

Sau khi tải về và giải nén ra, các bạn sẽ có những file và thư mục như thế này:


 

 

Chú thích các file:

  • Thư mục global, levels, movies => Thư mục chứa các file *GXT và các file *TXT đã xuất từ file *GXT của game.
  • Character manhunt2.txt => File chứa ký tự chuyển đổi thành tiếng việt.
  • Manhunt 2 VH.rar => Dữ liệu mẫu để bỏ vào game, nó chứa font và text đã được việt hóa.
  • Manhunt2 Text.txt => File Text txt được nối lại thành một file.
  • Manhunt2.xlsx => File Excel chứa dữ liệu text trong game, các bạn dịch ở file này.
  • Text Tool Manhunt 2 Alokill.exe => Tool để xuất nhập text các file *GXT trong các thư mục global, levels, movies.

 

Bước 2 - Tiến Hành Dịch Game

 

Các bạn mở file Excel Manhunt2.xlsx chứa text để việt hóa lên, giao diện của file đó như sau:

 

 

Ở tab Text như hình được chia ra làm 3 cột:

  • Cột B: Tiếng Anh, cột nguyên bản không nên chỉnh sửa ở cột này. Nó được dùng để tham khảo khi dịch.
  • Cột C: Máy Dịch, được dùng để tham khảo cho anh em nào muốn dịch.
  • Cột D: Tiếng Việt, cột quan trọng nhất, các bạn dịch chủ yếu ở cột này. Những chỗ nào không dịch nó sẽ tự động lấy dữ liệu bên cột Tiếng Anh để thay vào ở Tab Result.

 

 

Ở tab Result như hình trên. Các bạn quan tâm ở Cột A. Sau khi dịch được 1 câu bên tab Text nó sẽ tự chuyển qua tab Result này để xuất ra text dịch. Những câu nào chưa dịch thì nó sẽ lấy dữ liệu bên cột Tiếng Anh qua.

 

Sau khi dịch hết cột bên tiếng việt, hoặc các bạn muốn test thử những gì mình đã dịch trước đó. Chúng ta tiến hành bước tiếp theo là Xuất Text.

 

Bước 3 - Tiến Hành Xuất Text Đã Dịch

 

Ở những bài hướng dẫn trước mình có đính kèm bộ tool xử lý text của bác Matran999, nhưng mình cảm thấy quá lằng nhằng, nên mình quyết định publish nó lên Github để dễ dàng sử dụng online được luôn cho tiện. Các bạn hãy vào trang bên dưới để mở tool lên nhé.

 

À mình có chỉnh sửa lại tool này đôi chút để phù hợp với mình, nhưng sẽ không ảnh hưởng gì đâu. Các bạn không cần quan tâm đến nó làm gì.

 

https://alokillgtv.github.io/jointext/index.html

 

 

Sau khi mở tool lên, các bạn chọn Công Cụ Xử Lý Text. Sau đó các bạn chọn vào nút nhập ký tự ở cột bên phải màn hình. Một cửa sổ chọn file sẽ hiện lên. Các bạn hãy chọn lấy file Character manhunt2.txt trong thư mục dữ liệu mình đã cung cấp.

Nó sẽ tự chuyển các ký tự phù hợp.

 

Các bạn mở file Excel đã dịch lên, để xuất text các bạn mở Tab Result kế bên và chọn cột A và copy hết cột đó như hình bên dưới:

 

 

Sau khi copy, các bạn mở lại tool bác Matran999 copy text bên Excel sang khung text cần mã hóa, rồi nhấn nút mã hóa. Đợi một tí tool mã hóa xong thì copy hết text đã mã hóa. (Ai máy yếu sẽ hơi lag một tí vì hơi nhiều text, các bạn có thể dùng tool của _Cloud_ ở đây thay thế cũng được, chỉ cần ấn nút chuyển đổi copy các cột ký tự phù hợp là được. Do mình dùng quen tool của bác Matran999 hơn, nên hướng dẫn các bác dùng cái này, dù gì cũng thấy chuyên nghiệp hơn)

 

 

Sau khi copy text đã mã hóa ở tool xử lý text. Các bạn hãy dán nó hết vào file Manhunt2 Text.txt ở thư mục dữ liệu việt hóa mình đã cung cấp.

Text Tool Manhunt 2 Alokill.exe lên để tiến hành chèn text. Giao diện của nó như sau:

 

 

Các bạn hãy nhấn vào nút Nhập File để chèn các text vào file GXT của game. Một tiến trình sẽ hiện lên. Đợi chúng chạy xong sẽ có thông báo cho bạn.

 

 

Vậy là đã xong, tất cả text bạn dịch đã được chèn vào file *GXT.

 

Bước 6 - Copy Dữ Liệu Đã Dịch Và Test Bản Dịch

Vậy là đã xong xuôi mọi thứ, để test bản dịch. Trước hết các bạn hãy copy file Manhunt 2 VH.rar vào thư mục gốc của game. Sau đó giải nén nó ra để có bộ font đủ ký tự việt hóa. Các bạn có thể test thử xem trước.

 

Sau đó các bạn mở lại thư mục dữ liệu việt hóa. Copy 3 thư mục global, levels, movies đã được chèn text vào các file *GXT. Rồi dán 3 thư mục đó vào thư mục gốc của game.
