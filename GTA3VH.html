I. Giới Thiệu


 

Game này tuy rằng đã ra khá là lâu rồi, nhưng đến giờ vẫn chưa có bản việt hóa. Có lẽ là do nhiều người chưa biết cách làm, hoặc khó xử lý bộ font của game này, do đó không thể hiện được tiếng việt có dấu. Vì thế mình lập ra bài viết này để giúp các bạn có thể dịch game này dễ dàng. Bên cạnh đó mình sẽ đính kèm phiên bản được dịch hoàn toàn bằng tiếng việt, nhưng do trí tuệ nhân tạo AI dịch. Đọc cũng khá dễ hiểu mặc dù nhiều chỗ AI nó không hiểu ngữ cảnh nên dịch khá buồn cười. Nhưng trong lúc chờ đợi bản tiếng việt hoàn chỉnh của một cao nhân nào đó thì các bạn có thể tạm chơi phiên bản này cũng được.

 

Riêng bản thân mình rất gà english nên không thể tự dịch được, nhưng sẽ giúp các bạn các bược để có thể dịch được hoàn toàn tựa game này. Sau đây là một số hình ảnh demo mình làm tiếng việt cho nó.


 

 

 

 

 

 


 

Như các bạn thấy đó, bộ font này mình đã tùy chỉnh khá rõ và đẹp. Tuy nhiên nó cũng có nhược điểm. Nhược điểm lớn nhất của nó là nó không bao gồm bộ font tiếng việt viết hoa, vì game không hỗ trợ đủ ký tự để việt hóa được các ký tự viết hoa đó. Tuy nhiên cũng đã đủ để thể hiện toàn bộ tiếng việt trong game như các bạn thấy ở mấy tấm hình trên.

 

Và đây là bản AI dịch full game dành cho các bạn nào thích thì tải về và cài đặt để chơi:

 

GTA3 VH 1.0.zip

drive.google.com

 

Các bạn tải về bỏ vô trong thư mục gốc của game, và nhấn chuột phải chọn Extract Here, một thông báo hiện lên các bạn chọn Yes to All là xong. Vào game và chơi thôi.

 

II. Hướng Dẫn Dịch Game


 

Các công cụ cần thiết để dịch GTA3:

  • Notepad++ (Tự cài nhé bạn)
  • Excel
  • Gta3gxt.exe (Tool xuất text GTA 3, nó nằm trong file nén dữ liệu mình sẽ cung cấp bên dưới)
  • Bộ Tool hỗ trợ dịch game của bác Matran999 ở đây. (Mình cũng đính kèm sẵn trong file dữ liệu)

Bước 1: Tải các dữ liệu cần thiết để việt hóa

Các bạn hãy tải file nén chứa công cụ và dữ liệu việt hóa về máy.

 

Tool VH GTA3.zip

drive.google.com

 

Bên trong thư mục các bạn tải về có những thứ như sau:

 

  1. Thư mục tool của bác Matran999
  2. american.gxt (File Gxt là file chứa text của game, đây là file gốc của game)
  3. american.txt (File text đã xuất từ file GXT gốc của game, đây là file gốc chưa được việt hóa)
  4. GTA3 TEXT.xlsx (File excel mình đã chuyển từ file american.txt sang để các bạn dễ việt hóa và ít bị lỗi)
  5. GTA3 VH 1.0.zip (File nén các dữ liệu trong game bao gồm font và text đã việt hóa);
  6. gta3gxt.exe Tool để xuất và nhập text vào file american.gxt.
  7. Character GTA3.txt File chứa các ký tự để chuyển đổi sang tiếng việt.

Bước 2: Chép dữ liệu việt hóa vào thư mục game

 

1. Các bạn copy file GTA3 VH 1.0.zip vào thư mục gốc của game.

2. Nhấn chuột phải vào file và chọn Extract Here để xuất các dữ liệu vào game nó gồm font và text đã dịch. Và nhấn vào Yes To All.

 

 

Vậy là từ giờ trong thư mục game các bạn có các dữ liệu việt hóa cần thiết là font và text đã được chỉnh sửa để hiển thị đầy đủ tiếng việt cho nó.

 

Bước 3: Xuất text và tiến hành dịch game.

 

Mình đã làm sẵn text cho các bạn rồi, nên các bạn chỉ cần mở file GTA3 TEXT.xlsx lên để dịch ngay và luôn. Giao diện của file GTA3 TEXT.xlsx như sau.

 

 

4. File Excel này được chia làm 4 cột chính.

  • Cột A là cột ID, tuyệt đối không chỉnh sửa cột này.
  • Cột B là cột tiếng anh gốc của game. Các bạn cũng không nên sửa gì ở đây, để dễ đối chiếu.
  • Cột C là cột đã được AI dịch sẵn mang mục đích tham khảo cho bạn.
  • Cột D là cột chính, các bạn phải dịch ở cột này.

5. Quy tắc để dịch:

  • Các bạn dịch cần lưu ý các code của game, nó có dạng ~r~ hay ~h~~k~~PED_LOOKBEHIND~~w~, tuyệt đối phải giữ nguyên cấu trúc của nó.
  • Các bạn không được dịch tất cả bằng viết Hoa có dấu. VD: MÁY BAY TRỰC THĂNG mà phải dịch là Máy Bay Trực Thăng. Tuy rằng cho dù các bạn có dịch bằng viết Hoa có dấu thì khi chuyển ký tự nó cũng sẽ chuyển sang Máy Bay Trực Thăng như thế này thôi.
  • Không được để ô trống trong khung tiếng việt. Khi nào các bạn muốn xuất text thì cần phải sao chép các dữ liệu gốc tiếng anh hoặc dữ liệu AI dịch vào ô trống để tránh lỗi game.

6. Xuất text sau khi dịch xong:

Sau khi bạn dịch xong, các bạn cần phải làm theo các bước như sau để chuyển lại vào game. Những bạn nào biết các dùng Excel sẽ làm việc này rất rất dễ.

1. Các bạn cần phải copy các dữ liệu vào các ô trống còn lại, để kiểm tra các bạn làm như sau. Chọn cột D và nhấn vào Filter tắt hết các Select All.

 

 

2. Nhấn vào chọn ô Blank (trống) để hiện các ô trống còn chưa dịch như hình.

 

 

3. Các bạn nhấn vào ô trống rồi nhấn phím = rồi chọn cột tương ứng với hàng đó. Các bạn có thể chọn cột Eng hoặc cột AI dịch đều có thể được để sao chép chúng vào ô trống như hình.

 

 

4. Sau khi sao chép vào các ô còn trống. Các bạn bung các dữ liệu bị ra lại bằng cach vào filter ở cột D và chọn Select All như hình

 

 

5. Sau khi bung ra hết, các bạn cần ẩn hết các cột tiếng Anh và Cột AI dịch đi. Chỉ còn lại 2 cột A (ID), cột D (Cột các bạn dịch), để ẩn các bạn làm giống mình, chọn phần trên các cột B và nhấn chuột trái rồi nhấn vào dòng Hide để ẩn chúng đi.

 

6. Sau khi ẩn các cột đi các bạn sẽ còn lại 2 cột như hình.

 

 

7. Các bạn copy toàn bộ text ra một file txt, bắt đầu từ dòng A2 cho đến dòng D2650 ở cột D và nhấn CTRL + C để copy chúng.

 

 

8. Mở file american.txt lên bằng notepad++ rồi dán toàn bộ text vừa copy ở file Excel vào nó.

 

[/font][/size][/color]

 

9. Các bạn cần replace nó để phù hợp với quy tắc của file american.gxt cần. Ở notepad++ các bạn nhấn tổ hợp phím CTRL + H để mở trình Replace của notepad++ lên. Các bạn config như trong hình bên dưới, với ô Find What là \t và Replace With là (Đây là dấu khoảng cách nhé, chứ không phải ô trống đâu). Sau đó các bạn nhấn Replace All để replace hết file text.

 

[/font][/size][/color]

 

10. Và đây là thành quả.

 

 

11. Tiếp theo cần phải chuyển nó sang các ký tự có tiếng việt trong game. Các bạn vào thư mục Tool Support Matran999 và mở tiếp thư mục Join Text bên trong. Sau đó các bạn mở file index.html bằng Chrome trên máy tính. Sau đó các bạn chọn Công Cụ Xử Lý Text như hình bên dưới



 

 

12. Sau đó ở cột bên phải có nút Nhập Ký Tự, các bạn nhấn vào đó và chọn file Character GTA3.txt ở bên trong thư mục Tool VH GTA3 mà các bạn tải về, nó sẽ nhập các ký tự phù hợp vào cột này.

 


 

13. Và ở dòng ký tự đầu ra các bạn sẽ thấy dấu ##, các bạn giúp mình đổi thành dấu ". Vì lý do nào đó tool không nhận dấu này khi lưu nên các bạn phải thao tác bằng tay.


 

 

14. Lúc này các bạn copy các dữ liệu đã copy từ file Excel sang txt và bỏ vào nhập text của tool như hình bên dưới. Sau đó nhấn vào nút mã hóa để chuyển sang ký tự việt.

 


 

15. Sau khi tool chạy các ký tự việt sẽ chuyển thành các ký tự phù hợp để thể hiện tiếng việt trong game như hình. Sau đó các bạn copy tất cả text đã mã hóa.


 


 

16. Trong dữ liệu việt hóa tải về có phần mềm gta3gxt.exe, hãy mở nó lên và chọn file american.gxt để mở nó ra.


 


 

17. Các bạn nhấn vào text và dán tất cả các text đã mã hóa vào khung dữ liệu của tool gta3gxt.exe như hình bên dưới. Lưu ý, các bạn cần xóa dòng trống cuối cùng ở tool nhé, Để dòng trống sẽ bị báo lỗi không lưu được đó.

 


 

18. Vậy là xong, các bạn lưu file lại bằng File -> Save As. Sau đó copy file đó vào thư mục gốc trong game bằng đường dẫn GTA 3\TEXT\

 

 

19. Ok vậy là xong rồi, vào game test thử ngay và luôn nào. Nếu làm sai, các bạn xem kỹ lại các bước nhé, rồi sẽ làm được ngay thôi.
